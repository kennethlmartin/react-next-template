## Development

### Getting Started

1. `npm install`
1. `npm run start:dev`

## Production

### Build
> Generate production assets.

```bash
npm run build
```

### Sitemap
> Generate sitemap(s) and robots.txt for all static/pre-rendered pages.

```bash
npm run build:sitemap
```
